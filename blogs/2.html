What is blockchain?

Blockchain is the most secure database(a place where we store data. Eg- microsoft excel.) to ever exist. No, it’s not immune to hacking but it is decentralised which makes it impossible to hack. This may sound technical so let’s start with the very basics.

A blockchain as its name suggests contains blocks where data is stored. This data can be anything but mostly this data is transaction details. All these blocks are connected with each other l by a digital chain, hence blockchain.

Now let’s get a step further and understand how this blockchain is secure and how it works. With the information in each block, the block also contains a hash code of it’s own and the hash code of the previous block. This hash code is unique and is generated using the components of the block. For example this is the hash if the block contains the word

Expect: 5312FB609F60384731FCFCB95DEEF3602239BF61F865A07BD8E08D818D22E9FA
Aspect:0D608D98EBC6BB93C2425F13680A999BBE812FE3B4DF75D2EBC0AF7FC1440F59

So each block hash code is completely unique and is according to the data it stores. Also the block has the hash of the previous block also attached to it. So this connects the blocks together forming a chain. Now this chain is unbreakable as if someone tampers with the data in a block it’s hash code will change which will result in breaking of the chain as the hash code will not match the previous block hash code present in the next block. This is how the data is secured. This is level one. But what makes the blockchain truly secure and unique is it’s decentralisation.

Decentralisation is the power from one major organisation to masses. So instead of Goldman Sachs or Bank of America maintaining our transaction databases, it’s us: the people.

So each copy of the complete blockchain is freely accessible to everyone. Each person can have the whole blockchain on his computer and verify the transactions. When a new block is added to the blockchain, it is sent to everyone who has the blockchain and everyone verifies that things are in the order and the block is not tampered with using code and when everything is checked out, ech node(people having access to the blockchain) adds that block to the blockchain.

This is the highest level of security with decentralisation. So instead of facebook managing our data and adding and removing pictures unilaterally, we can do that in a public database: blockchain. Instead of banks keeping track of our transitions and having complete control over our money, we can do it ourselves.

TL;DR

1.Blockchain is just a database, just very secure.
2. Anything can be stored in a blockchain, cryptocurrencies uses it for storing ledgers(transaction records)
3. Blockchains are decentralized. Insted of a single organisation having access over data we all have it. (This is not private data about specific someone, this is genaral data.)
